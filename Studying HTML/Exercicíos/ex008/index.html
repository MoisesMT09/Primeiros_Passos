Principais Formatações

Negrito / Destaque

Nesta frase temos uma este campo em negrito usando a tag b(Não semantico)

Nesta frase temos uma este campo em destaque usando a tag strong.(semantico)

Italico / Enfase

Nesta frase temos este campo em Italico usando a tag i.(Não semantico).

Nesta frase temos este termo em enfase usando a tag EM.(semantico)

Texto grande e Pequeno

Texto Excluído

Podemos usar uma tag para mostrar que um texto foi excluido e já não deve ser considerado.

Texto Inserido

Podemos inserir um textos e marcar ele para simbolizar que foi adicionado depois.

Existe também a tag U que é o sublinhado(não semantico)

Podemos representar texto que os queremos po cima de outros. Ex.: x3+3x2+1

Texto Subscrito

Podemos escrever txtos que queremos coloca-los abaixo de outros. Ex.: H2O.
